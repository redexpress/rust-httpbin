# Roadmap

## v0.1.0 — First Release ✅

- [x] Request echo endpoints (GET/POST/PUT/PATCH/DELETE)
- [x] Header / IP / User-Agent inspection
- [x] Status code endpoint
- [x] Delay endpoint
- [x] Redirect endpoint
- [x] Stream endpoint (SSE)
- [x] Basic auth endpoint
- [x] Bearer token endpoint
- [x] UUID generation
- [x] Catch-all `/anything` endpoint
- [x] OpenAPI + Swagger UI
- [x] Request ID middleware
- [x] Access log middleware
- [x] Tracing spans

## v0.2.0 — Planned

- [x] Gzip / Brotli compression
- [x] Response headers endpoint (`/response-headers`)
- [x] Cookies endpoint (`/cookies`)
- [x] Image endpoints (`/image`, `/image/png`, …)
- [x] Form-encoded data parsing in echo endpoints
- [x] Multipart file upload support

## v1.0.0 — Future

- [ ] HTTP/2 support
- [ ] Prometheus metrics
- [ ] Rate limiting
- [ ] WebSocket testing endpoint
- [ ] SSE example endpoint improvements
- [ ] Docker image
